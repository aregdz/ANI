# АНИ — Альбом народных историй

Полноценный Django-проект: карта мира, фильтр по датам, регистрация и вход по телефону, добавление историй с фото/видео/аудио, модерация администратором, удаление/редактирование историй и пользователей.

## Запуск

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py create_admin_owner --phone +79990000000 --password +79990000000
python manage.py runserver
```

Открыть: http://127.0.0.1:8000/

## Администратор

По умолчанию один администратор:

- телефон: `+79990000000`
- пароль: `+79990000000`

Можно изменить через переменные окружения или команду `create_admin_owner`.

## Логика

- Пользователь регистрируется без подтверждения.
- Логин — телефон, пароль задается при регистрации.
- Истории обычных пользователей получают статус «На подтверждении».
- Администратор видит истории на подтверждение и может опубликовать, редактировать или удалить.
- Опубликованные истории появляются на карте.
- Администратор может удалять пользователей, кроме главного администратора.

## Структура

- `stories/models.py` — пользователь и история.
- `stories/forms.py` — формы регистрации, входа, истории.
- `stories/views.py` — вся бизнес-логика.
- `templates/stories/` — HTML-шаблоны.
- `static/stories/css/style.css` — оформление.
- `media/` — загруженные файлы после запуска.

## Карта

Используется Leaflet и OpenStreetMap через CDN. Для продакшена можно скачать ассеты локально.

## Карта
В проекте используется Яндекс.Карты JavaScript API с ключом, указанным в `templates/stories/base.html`.
Если нужно заменить ключ, поменяйте значение `apikey` в строке:
`https://api-maps.yandex.ru/2.1/?apikey=...&lang=ru_RU`
